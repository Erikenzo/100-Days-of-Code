This project uses functions and OOP.

Main file contains main program - Coffee machine. You have 3 options latte, espresso and cappuccino for coffees, after you enter name of coffee you need to insert coins and input value of each coin how many of those you drop. If enough coins prints coffee with charge and etc. if not shows that it is not enough. 

There are 2 more commands if for What would you like? We will type report will show us money which machine has and resources like coffee, milk and water, quit command to turn off machine.
